This project analyzes and visualizes **38,000+ consumer loan records** to uncover approval patterns, borrower risk profiles, and repayment performance. The dashboard is built entirely in Excel, designed for fast, data-driven decision-making.

---

## üîç Key Objectives
- Summarize borrower and loan characteristics in a clear format  
- Build logic-based KPIs around credit risk and portfolio quality  
- Segment customers by income, grade, term, verification status, and DTI  
- Provide an interactive dashboard for instant filtering and insights

- **Excel**
- **Pivot Tables**
- **Power Query (optional cleanup)**
- **Lookup Functions (VLOOKUP, XLOOKUP, INDEX-MATCH)**
- **Conditional Formatting**
- **Slicers & Interactive Charts**

---

## üìà Key Features

### **1. Interactive Dashboard**
- Approval and repayment trend visuals  
- Borrower demographics: grade, sub-grade, home ownership, employment length  
- Loan metrics: interest rate, DTI, installment, loan amount  
- Quick slicers for filters such as **grade**, **term**, **verification status**, and **income group**

---

### **2. Logic-Based KPIs**
- **Default Rate** (using payment/delinquency flags)  
- **Approval Ratio**  
- **Income Band Distribution**  
- **Credit Utilization Indicators** (combining loan_amount, total_acc, and payment metrics)

These KPIs help pinpoint high-risk borrower segments and weak areas in the portfolio.

---

### **3. Data Structuring for Analysis**
- Cleaned and organized 25+ columns of borrower + loan attributes  
- Created derived fields such as:
  - Income groups  
  - Risk bands based on grade/sub-grade  
  - EMI-to-income impact  
- Used pivot tables to summarize:
  - Loan amount distribution  
  - Installment patterns  
  - DTI categories  
  - Verification vs. performance relationships  

---

### **4. Insightful Visualizations**
- Term-wise and grade-wise performance charts  
- Interest rate and DTI trend visuals  
- Category-wise borrower breakdowns  
- Slicers for interactive exploration  

---

## üßæ Outcome
The final Excel dashboard transforms raw loan entries into **clear, business-ready insights**, enabling faster evaluation of risk, customer behaviour, and segment performance.
